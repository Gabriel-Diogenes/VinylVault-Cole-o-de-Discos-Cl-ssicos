Sobre o VinylVault

VinylVault é um acervo digital dedicado aos álbuns mais influentes da história da música. De jazz clássico a hip-hop nacional, cada disco conta uma história que vai além das faixas — uma janela para o contexto cultural, social e artístico de sua época.

O projeto foi desenvolvido como aplicação web dinâmica, com dados servidos por uma API REST simulada via JSON Server. As informações são carregadas e renderizadas por JavaScript puro, sem frameworks, priorizando boas práticas de HTML semântico, CSS moderno e responsividade.
